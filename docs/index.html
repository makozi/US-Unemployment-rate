US Unemployment rate from 1948 to August 2016¶

Using unemployment data set from the Federal Reserve Bank of St. Louis . The data set can be downloaded from this link

Needs to convert this column to the datetime type using the pandas.to_datetime() function, which returns a Series object with the datetime data type that we can assign back to the DataFrame

import pandas as pd
import matplotlib.pyplot as plt
unrate=pd.read_csv('unrate.csv')
unrate['DATE']=pd.to_datetime(unrate['DATE'])
print(unrate.head(12))

While the y-axis looks fine, the x-axis tick labels are too close together and are unreadable.

The xticks() function within pyplot lets you customize the behavior of the x-axis ticks. Check out this documentation for that function.

In [8]:
plt.plot(first_twelve['DATE'], first_twelve['VALUE'])
plt.xticks(rotation=90)
plt.show()

We add label for x and y axis and also title to the plot

plt.plot(first_twelve['DATE'], first_twelve['VALUE'])
plt.xticks(rotation=90)
plt.xlabel('Month')
plt.ylabel('Unemployment Rate')
plt.title('Monthly Unemployment Trends, 1948')
plt.show()
